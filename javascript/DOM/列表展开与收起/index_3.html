“烧烤”天气带热亲水目的地 海岛游、反季游订单猛增

2018年07月06日 16:03 来源：人民网 作者：田虎

近日，全国多地迎来酷暑天气，不少游客都开始选择逃离“火炉”般的城市，通过出游来避暑。旅行社方面表示，目前一些亲水目的地比较受到游客青睐，同时海岛游、反季游的订单有明显增长。

水上乐园或持续出现“下饺子”景象

随着天气持续炎热，亲水等主题游受市场关注度越来越高，搜索、预订量激增。驴妈妈旅游网提供的数据显示，今年暑期，上海迪士尼乐园、峨眉山、黄山风景区、北京欢乐谷、珠海长隆海洋王国、苏州乐园森林水世界、广州长隆水上乐园、张家界天门山国家森林公园、冒险岛水世界、上海玛雅海滩水公园成为十大热门景区，其中水上乐园占据四席，成为暑期“爆款”。

业内人士指出，对于国内夏季动辄35℃以上的高温“烧烤”天气，玩水“湿身”无疑是最佳避暑方式。驴妈妈相关负责人指出，今年水上乐园门票预订人次较去年翻番，暑期或将持续出现“下饺子”的壮观景象。此外，峡谷漂流也是许多游客夏季避暑的“心头好”，如安吉大溪峡谷皮筏漂流、杭州临安龙井峡漂流、武隆仙女峡漂流、清远黄腾峡漂流等景区预订量迅速攀升。

海岛游订单增长明显

除了水上乐园以外，海岛游也是游客今年暑期出游的主要方式。驴妈妈相关负责人表示，海岛避暑依然是暑期最热门的出境游主题之一，普吉岛、岘港、巴厘岛、毛里求斯等热门海岛游客预订热情高涨。而从携程提供的数据来看，普吉岛、新加坡、巴厘岛、哥打京拿巴鲁的暑期酒店订单量同比均有明显提升，巴厘岛订单提升十分明显，同比增幅达到44%。同时，携程方面表示，东南亚海岛成为了最受亲子用户欢迎的暑期游圣地，普吉岛、巴厘岛、哥打京拿巴鲁、甲米、芭堤雅、兰卡威、苏梅岛等都在亲子酒店TOP10热门城市之列。其中，哥打京那巴鲁和芭堤雅的订单量同比增长近50%。

此外，北京首都航空相关负责人透露，为了满足更多游客对海岛游的需求，目前首都航空还推出了北京=马累航线的优惠乘机服务，每周三班，往返3500元（不含税）起，航班适用日期为2018年7月1日起至2018年12月21日。

同程旅游相关负责人表示，海岛因其气候宜人、环境优美等，加上大多海岛对中国游客实行免签或落地签政策，非常适合和家人、闺蜜、同学等一起休闲度假，因此也成为人们今夏避暑度假的首选，像巴厘岛、沙巴、普吉岛、宿雾、塞班岛等地都是近期咨询和预订的热点。

“反季节目的地”成游客新宠

相比亲水类目的地，也有一些游客愿意选择南半球或国内西部各省这种反季节目的地作为避暑方式。“北半球的夏季正值南半球的冬季，不少消费者选择以反季节出游的方式来消夏避暑，而且价格相较于澳洲旺季也便宜不少，性价比较高。”同程艺龙相关负责人表示，维珍澳大利亚航空于7月2日开通香港直飞悉尼航线，同程艺龙也联合维珍澳航推出了众多香港飞往墨尔本、布里斯班、凯恩斯等热门旅游目的地的国际优惠航班。“每年6-8月我们正处严夏，澳洲刚进入冬季，但是并不寒冷，加上澳洲拥有优美的自然风光和珍稀动植物资源，相信新航线开通后将吸引更多的中国游客前往澳洲旅游。”

同时，记者了解到，国内西部一些省市因气候凉爽，也成为反季游的热门选择。其中，有“中国避暑之都”之称的贵州更是在7月16日至9月30日间，面向江浙沪、两广、两湖等10个省(区、市)推出包括景区门票5折、高速通行5折等优惠政策。此外，驴妈妈相关负责人表示，针对暑期避暑出游需求，目前也围绕呼伦贝尔、额尔古纳、香格里拉等目的地打造了众多个性化跟团线路，而像丽江、银川等西南、西北避暑胜地今年也都非常受游客追捧。
